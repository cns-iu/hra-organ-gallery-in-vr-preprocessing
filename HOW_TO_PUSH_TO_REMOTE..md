To push `main` to public repo at https://github.com/cns-iu/hra-organ-gallery-in-vr-preprocessing.git

1. Add remote:
``` {bash}
git remote add production https://github.com/cns-iu/hra-organ-gallery-in-vr-preprocessing.git
```
2. Push the `main` branch to production: 
``` {bash}
git push production main  
```
